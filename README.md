# vscode-ext-phpeditinfo
* vscode插件，保存PHP文件时记录作者与时间

# 首次使用
* 在out和src目录下创建config.json
```
{
  "userName" : ""
}
```
* 点击ctrl+shift+p或F1，输入set user 在弹出的输入框输入用户名。
![image](https://github.com/NateHuangHao/vscode-ext-phpeditinfo/tree/master/gif/show1.gif)

# 使用效果
* 编辑完后按ctrl+s保存
